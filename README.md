# BLOCK DROP

オフラインでも動くブロックパズルゲーム（テトリス風、著作権を侵害しないオリジナルデザイン）

🎮 **プレイする**: https://あなたのGitHubID.github.io/block-drop/

## 特徴

- 完全オフライン動作（HTML単体ファイル、外部依存なし）
- 6種類のゲームモード（CLASSIC / LEVEL FIXED / RISING / NO BAR / BOMB / STAGE）
- STAGEモード：20ステージのチャレンジ、全クリアでエンディング演出
- スマホ対応のタッチ操作（3段ボタン配置）
- BGM・効果音（Web Audio APIで生成、外部音源不使用）
- スコアランキング・プレイ履歴をブラウザに保存
- ホーム画面に追加してアプリのように利用可能（PWA対応）
